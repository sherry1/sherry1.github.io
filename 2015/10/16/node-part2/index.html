node.js学习笔记——nodejs核心模块（二）

  • nodejs

本章内容主要包括：全局对象，常用工具，事件机制，文件系统访问，http服务器与客户端

全局对象（process，console）

全局对象：nodejs中的全局对象是global，global的最根本的作用是作为全局变量的宿主。

全局变量：所有全局变量都是global的对象属性。如console、process，以下一一介绍下。
1、process
定义：用于描述当前nodejs进程状态的对象，提供了一个与操作系统的简单接口。
process.argv:命令行参数数组（第一元素node,第二个元素脚本文件名，第三个参数以后就都是运行参数）

process.stdout：标准输出流
process.stdin：标准输入流
process.nextTick(callback):作用是nodejs会在下次事件循环响应时调用callback，把耗时操作拆分为两个事件，减少每个事件的执行时间，提高响应速度。

2、console:略

常用工具util

定义：提供常用函数的集合，用于弥补核心js的功能过于精简的不足。

1、util.inherits（constructor, superConstructor）
概述：是一个实现对象间原型继承的函数。原型可以扩展构造函数，可以改变构造函数中的属性，用util.inherits只可以继承原型中的属性，却不能继承原型中构造函数的属性。

function Base() {
    this.name = 'base';
    this.base = 1991;
    this.sayHello = function() {
        console.log('Hello ' + this.name);
    };
}
Base.prototype.showName = function() {
    console.log(this.name);
};
function Sub() {
    this.name = 'sub';
}
util.inherits(Sub, Base);  //Sub继承Base
var objBase = new Base();
objBase.showName();        //base
objBase.sayHello();        //Hello base
console.log(objBase);    //{ name: 'base', base: 1991, sayHello: [Function] }

var objSub = new Sub();
objSub.showName();        //sub
// objSub.sayHello();    //报错
console.log(objSub);    //{ name: 'sub' }

2、util.inspect（object,showHidden,depth,[colors]）
概述：是一个将任意对象转换为字符串的方法，至少接受一个参数，即要转换的对象。
object:要转换的对象

除了上述函数以外，util还提供了很多，可以访问http://nodejs.org/api/util.html学习

事件驱动events

nodejs最重要的模块
1、事件发射器
概述：
events模块只提供一个对象,events.EventEmitter。EventEmitter的核心就是事件发射和事件监听器的封装。
过程：
EventEmitter事件发射（事件名[字符串]，参数1，参数2….），对于每一个事件，EventEmitter支持若干个事件监听器，当事件发射时，注册到这个时间的事件监听器被依次调用，事件参数作为回调函数参数传递。
EventEmitter常用API：
EventEmitter.on(event,listener)为指定事件注册个监听器
EventEmitter.emit(event,[arg1],[arg2]…)发射event事件，传递若干可选参数到事件监听器的回调函数的参数中
EventEmitter.once(event,listener)为指定事件注册一个单次监听器，触发一次后立即解除该监听器。
EventEmitter.removeListener(event,listener)移除指定事件的某个监听器，listener必须是该事件已经注册过的监听器。
EventEmitter.removeAllListener([event])移除所有事件的所有监听器，若指定event，则移除指定事件的所有监听器。

var events = require('events');
var emitter = new events.EventEmitter();
emitter.on('someEvent', function(arg1, arg2) {
console.log('listener1', arg1, arg2);
});
emitter.on('someEvent', function(arg1, arg2) {
console.log('listener2', arg1, arg2);
});
emitter.emit('someEvent', 'byvoid', 1991);

//运行结果 
listener1 byvoid 1991
listener2 byvoid 1991

2、error事件
我们一般要为会发射error事件的对象设置监听器，避免遇到错误后整个程序崩溃。

var events = require('events');
var emitter = new events.EventEmitter();
emitter.emit('error');

3、继承EventEmitter
大多时候我们不会直接使用EventEmitter，而是在对象中继承它。包括fs,net,http在内的，只要是支持事件响应的核心模块都是EventEmitter的子类。

文件系统fs

概述：fs模块是文件操作的封装。塔提供了文件的读取，写入，更名，删除，遍历目录，链接等。所有操作都提供异步和同步两个版本。
1、fs.readFile(filename,[encoding],[callback(err,data)])

var fs = require('fs');

//不指定编码
fs.readFile('content.txt', function(err, data) {
    if (err) {
        console.error(err);
    } else {
        console.log(data);   //运行结果<Buffer 54 65 78 74 20 e6 96 87 e6 9c ac e6 96 87 e4 bb b6 e7 a4 ba e4 be 8b>
    }
});

//指定编码
fs.readFile('content.txt', 'utf-8',function(err, data) {
    if (err) {
        console.error(err);
    } else {
        console.log(data);   //运行结果 Text 文本文件示例
    }
});

//若有错误发生，err将会是Error对象

2、fs.readFileSync(filename, [encoding])
读取到的文件美容会以函数返回值的形式返回。如有错误繁盛，fs将会抛出异常，你需要使用try和catch捕捉并处理异常。

3、fs.open(path, flags, [mode], [callback(err, fd)])
path必选，文件路径
flags必选，有r,r+,w,w+,a,a+等取值
mode可选，给文件指定权限

4、fs.read(fd, buffer, offset, length, position, [callback(err, bytesRead,buffer)])
一般来说，除非必要，否则不要使用这种方式读取文件

HTTP服务器与客户端

概述：http.server是一个基于事件的HTTP服务器，它的核心由nodejs下层C++部分实现，接口由js封装。
http.request是一个HTTP客户端工具，用户向服务器发起请求

一、HTTP服务器

1、http.server的事件
http.Server是一个基于事件的http服务器，所有请求都被封装成独立事件，我们只需要对它的事件编写响应函数即可实现http服务器的所有功能。
request（最常用）：客户端请求到来时，该事件被触发。两个参数req——http.ServerRequest的实例,res——ServerResponse的实例,表示请求和响应信息。
http提供的捷径：http.createServer([requestListener])，requestListener作为request事件的监听函数。

var http = require('http');
var server = new http.Server();
server.on('request', function(req, res) {
    res.writeHead(200, {'Content-Type': 'text/html'});
    res.write('<h1>Node.js</h1>');
    res.end('<p>Hello World</p>');
});
server.listen(3000);
console.log("HTTP server is listening at port 3000.");

connection:当TCP连接建立时，该事件被触发。一个参数socket——net.Socke的实例。
close:当服务器关闭时，该事件被触发。注意不是在用户连接断开时。

2、http.ServerRequest
是http请求的信息，它提供了3个事件用于控制请求体传输。
data、end、close

3、获取GET请求内容

var http = require('http');
var url = require('url');
var util = require('util');
http.createServer(function(req, res) {
    res.writeHead(200, {'Content-Type': 'text/plain'});
    res.end(util.inspect(url.parse(req.url, true)));
}).listen(3000);

4、获取POST请求内容

var http = require('http');
var querystring = require('querystring');
var util = require('util');
http.createServer(function(req, res) {
    var post = '';
    req.on('data', function(chunk) {
        post += chunk;
    });
    req.on('end', function() {
        post = querystring.parse(post);
        res.end(util.inspect(post));
    });
}).listen(3000);

5、http.ServerResponse
是返回给客户端的信息，决定了用户最终看到的结果。request事件的第二个参数。有三个重要的函数组成。

  • response.writeHead(statusCode, [headers])
    返回响应头，其中statusCode是http状态码，[headers]数组对象，表示响应头的每个属性。
  • response.write(data, [encoding])
    返回的相应内容，data是Buffer字符串，[encoding]是编码方式。在res.end之前可以多次被调用。
  • response.end([data], [encoding])
    表示响应结束，发送已经完成。必须被调用一次。如果不调用，客户端将永远处于等待状态。

二、HTTP客户端

下面是一个通过http.request发送POST请求的代码

var http = require('http');
var querystring = require('querystring');
var contents = querystring.stringify({
    name: 'byvoid',
    email: 'byvoid@byvoid.com',
    address: 'Zijing 2#, Tsinghua University',
});
var options = {
    host: 'www.byvoid.com',
    path: '/application/node/post.php',
    method: 'POST',
    headers: {
        'Content-Type': 'application/x-www-form-urlencoded',
        'Content-Length' : contents.length
    }
};
var req = http.request(options, function(res) {
    res.setEncoding('utf8');
    res.on('data', function (data) {
        console.log(data);
    });
});
req.write(contents);
req.end();
//运行结果
array(3) {
    ["name"]=>
    string(6) "byvoid"
    ["email"]=>
    string(17) "byvoid@byvoid.com"
    ["address"]=>
    string(30) "Zijing 2#, Tsinghua University"
}
