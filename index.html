js知识补充整理

stop()

概述：是jQuery中用于控制页面动画效果的方法。运行之后立刻结束当前页面上的动画效果。
stop在新版jQuery中添加了2个参数：

  • 第一个参数的意思是是否清空动画序列，也就是stop的是当前元素的动画效果还是停止后面附带的所有动画效果，一般为false，跳过当前动画效果，执行下一个动画效果；
  • 第二个参数是是否将当前动画效果执行到最后，意思就是停止当前动画的时候动画效果刚刚执行了一般，这个时候想要的是动画执行之后的效果，那么这个参数就为true。否则动画效果就会停在stop执行的时候

substr,substring,slice,split,join的使用

一、substr
语法：stringObject.substr(start,length)
start 可为负数，-1指倒数第一位。length 可选
二、substring
语法：stringObject.substring(start,stop)
start 必需，非负。stop可选，非负。
三、slice
语法：arrayObject.slice(start,end)
start 必需，同substr。end 可选，是结束处的数组下标，可为负，负数即是从尾部开始
四、split
语法：stringObject.split(separator,howmany)
用于把一个字符串分割成字符串数组。
separator 必需，为字符串或正则表达式。howmany 可选，返回数组的最大长度。
五、join
语法：arrayObject.join(separator)
用于吧数组中的所有元素合并成一个字符串。
separator 表示将数组元素连接起来的分隔符。

substring,slice都是从start开始（包括start），从stop或者end结束（不包括其本身）
